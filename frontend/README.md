# World-News

Временно размещено на тестовом сервере: http://w95004uj.bget.ru/

Использовано:
- React 16.5.2
- React-Router 4
- React-Redux 5
- Redux 4
- Webpack 4
- Babel 7

## Getting Started
Установка проекта в папке frontend
```sh
$ npm install
```

Запуск dev сервера
```sh
$ npm run dev
```
Основные маршруты:
  - localhost:8080/ - Главная
